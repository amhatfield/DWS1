HtML Element Definitions

<h1>
This is used for the Main heading.
<h2>
This is used for the sub heading.
<p>
This is used to the paragraph.
<b>
This is used to make things bold.
<i>
This is used to make thing italic.
<sup>
This is used to show things in superscript like dates.
<sub>
This is used to show things in subscript.
<br />
This is used to have line breaks.
<hr />
This is used to have horizontal breaks.
<strong>
This shows the content in a strong importance.
<em>
This gives an emphasis.
<blockquote>
Used when you have a quote that takes up most of the paragraph.
<q>
Used when you have a quote.
<abbr>
Used to show an abbreviation.
<cite>
Used this when you have to cite a source.
<dfn>
Used this when you need to explain a new term.
<address>
This show the contact info of the author of the page.
<ins>
This will insert things in to page.
<del>
This will delete things from the page.
<s>
This will make something relevant or not accurate.
<ol>
Used this when you need to make an order list.
<li>
Use this when you need to list something.
<ul>
Use this when you need to make an unordered list.
<dl>
Use this when you need to list definitions.
<dt>
This is used so show the term that you are defining.
<dd>
This is used when you are definitions for the terms.
